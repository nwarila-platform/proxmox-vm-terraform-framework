# Support

## Supported use case

This repository supports the Terraform framework itself, the repository automation around it, and the developer workflow needed to work on it safely and consistently.

## Out of scope

- Troubleshooting your specific Proxmox environment.
- Recovering failed infrastructure changes in downstream environments.
- Handling credentials or environment-specific secret management for consumers.

## When requesting help

Include:

- the command you ran
- the operating system
- the relevant workflow or script
- the exact error output
- whether the issue reproduces inside the devcontainer
